
# platfrom-ui

The frontend codebase of [Praromvik](https://github.com/Praromvik/praromvik).

## Build Setup

**Requires Node.js 16+**

```bash
# install dependencies
npm install

# serve in dev mode, with hot reload at localhost:5173
npm run dev

# build for production
npm run build

# serve in production mode
npm  run  preview
```
